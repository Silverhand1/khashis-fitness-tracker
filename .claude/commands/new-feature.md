# Add a New Feature

When asked to add a feature to the tracker:

1. Read `src/App.jsx` to understand current structure
2. Identify which tab it belongs to (today / workout / calories / history) or if it needs a new tab
3. Follow the existing style: dark bg, lime green accents, Barlow Condensed + DM Mono fonts
4. Update localStorage schema if new data needs to be saved
5. Test on mobile viewport (375px width)
